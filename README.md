# Designing-a-Sales-dashboard-in-Excel

Objective:
The objective of this project was to design a sales dashboard in Excel that analyses the sales data for different product categories. The company wanted to add user control for the product category so that users can select a category and see the trend month-wise and product-wise accordingly.

Dataset:
The dataset used for this project was the Sample E-Commerce database, which contained sales data for different product categories. The dataset included the following features: Order ID, Order Date, Ship Date, Aging, Ship Mode, Product Category, Product, Sales, Quantity, Discount, Profit, Shipping Cost, Order Priority, Customer ID, Customer Name, City, State, Country, Region, and Months.
